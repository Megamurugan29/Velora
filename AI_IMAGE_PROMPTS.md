## 📋 Usage Instructions

### For DALL-E 3:
1. Copy the full prompt under "DALL-E 3 / Adobe Firefly" section
2. Paste into DALL-E chat
3. Specify quantity: 1 (for single product) or 6 (for all products)
4. Download generated images
5. Rename according to convention

### For Midjourney:
1. Copy the prompt under "Midjourney Format"
2. In Discord, use: `/imagine prompt: [paste prompt here]`
3. Select best variation
4. Upscale to full resolution
5. Download image

### For Stable Diffusion:
1. Use the Stable Diffusion format prompts
2. Set resolution: 800x1000 for main images, 600x600 for details
3. Set sampling steps: 25-30
4. Set guidance scale: 7-12
5. Generate and download

### For Adobe Firefly:
1. Copy the detailed prompt
2. Enter in Adobe Firefly web interface
3. Adjust as needed for style
4. Generate and download

## 💾 Saving Images

1. Generate images using prompts above
2. Export as JPG (80-85% quality)
3. Name according to convention: `PROD-###-{Type}.jpg`
4. Save to correct folder:
   - Main: `images/products/main/`
   - Details: `images/products/details/`
   - Embroidery: `images/products/embroidery/`
5. Push to GitHub
