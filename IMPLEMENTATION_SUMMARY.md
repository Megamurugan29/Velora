## 📞 SUPPORT & CUSTOMIZATION

### To Change Admin Credentials
Edit `admin/admin.js`:
```javascript
const ADMIN_EMAIL = 'YourEmail@gmail.com';
const ADMIN_PASSWORD = 'YourPassword';
```

### To Add New Product Categories
1. Edit filter options in `shop/products.html`
2. Update `js/products.js` filter logic
3. Add category styling in `styles/shop.css`

### To Modify Color Scheme
1. Edit `style.css` CSS variables
2. Update accent colors in component CSS

### To Add Email Notifications
1. Implement email service (SendGrid, Mailgun, etc.)
2. Call API from `js/checkout.js` on order completion
